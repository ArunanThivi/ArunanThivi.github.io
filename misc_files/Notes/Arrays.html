Arrays

An array is a structured container whose components are:
  • a fixed length
  • A sequence of length containers of the same type numbered from 0
  • (Length field usually implicit in diagrams)
Arrays are anonymous, like other structured containers

Always referred to with pointers

For array A
  • Length is A.length
  • Numbered component i is A[i] (where i is the index)
  • IMPORTANT FEATURE: index can be ANY INTEGER EXPRESSION

Instantiating an array: 
Initializing an empty array: type[] name = new type [length]; 
Default value for integer arrays: 0
Default value for String arrays: null
Initalizing a non-empty array: type[] name = {val, val, val, val}
Trying to obtain array values before initializing the length throws an Array.IndexOutOfBounds error
Assigning an array to a new variable copies the pointer
Manipulating the array will change it for BOTH variables

public static int[] merge(int[] A, int[]B){
    int[] C = new int[A.length+B.length];
    int L0, L1;
    L0 = 0;
    L1 = 0;
    while (L0+L1 < C.length){
        if (L1 >=B.length || (L0 < A.length && A[0] < B[L1])){
            C[L0+L1] = A[L0];
            L0++;
        }
        else{
            C[L0+L1] = B[L1];
            L1++;
        }
    }
    return C;
}
