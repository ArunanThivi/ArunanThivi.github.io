Aide: How to Write a Loop
  1. Try to make a description of how things work on any arbritray iteration of the loop <- This is known as a loop invariant, because it is ALWAYS true 
  2. The loop body then MUST
    1. Start from any siutation consistent with the invatant ^ and condition
    2. Make progress in such a way as to make the invariant true again
  3. If our loops get the desired answer whenever the invariant is true, and the condition is false, our job is done!
